Technical Implementation

Seven AI is built on Botpress, an open-source conversational AI platform, with custom NLU integration for healthcare-specific entity recognition.

Core Architecture

1

Natural Language Understanding

Custom-trained NLU model using Botpress's engine with healthcare-specific entities and intents. The model recognizes symptoms, medications, and conditions from user input.

2

Conversation Flow

Dialog management using Botpress flows with conditional transitions based on extracted entities and user responses.

3

Business Logic

JavaScript code nodes handle symptom analysis, basic triage logic, and API integrations with medical databases.

NLU Implementation

The natural language understanding system is trained to recognize healthcare-specific entities:

Symptoms

Recognizes over 500 common symptoms and their variations (e.g., "headache", "nausea", "shortness of breath")

Medications

Identifies medication names, dosages, and frequencies in user input

Conditions

Detects mentions of medical conditions and comorbidities

Code Node Implementation

JavaScript code nodes handle the business logic for symptom analysis and response generation:

// Basic symptom analysis in a Botpress code node
const analyzeSymptoms = (entities) => {
  const symptoms = entities.filter(e => e.type === 'symptom');

  // Check for emergency symptoms
  const emergencySymptoms = ['chest pain', 'difficulty breathing', 'severe bleeding'];
  if (symptoms.some(s => emergencySymptoms.includes(s.value))) {
    return { severity: 'emergency', message: 'Please seek immediate medical attention' };
  }

  // Return analysis result
  return { severity: 'non-emergency', symptoms: symptoms.map(s => s.value) };
};

API Integration Example

// Fetching medication info from external API
const getMedicationInfo = async (medicationName) => {
  try {
    const response = await axios.get(`https://api.medicalinfo.com/meds/${medicationName}`);
    return response.data;
  } catch (error) {
    return { error: 'Medication not found' };
  }
};
